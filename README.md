# 💰 MoneyMind — Finance Dashboard

A modern fintech dashboard built with React and Tailwind CSS to track income, expenses, and financial insights in a clean and intuitive UI.

---

## 🚀 Live Demo

👉 https://your-vercel-link.vercel.app

---

## 📌 Features

* 🔐 Login system (UI-based)
* 📊 Interactive charts (Pie + Line)
* 💳 Financial summary cards
* 📁 Sidebar navigation
* 🌐 Currency selection (INR / USD)
* 👤 Role-based UI (Admin / Viewer)
* 🎨 Premium dark UI (glassmorphism)

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 📈 Recharts
* 🔀 React Router DOM
* 🎯 Lucide Icons

## ⚙️ Installation

Clone the project:

```bash
git clone https://github.com/your-username/moneymind-dashboard.git
```

Go to project folder:

```bash
cd moneymind-dashboard
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
